# Demo 2: Reachy Mini 滑块控制

这是一个通过 tkinter GUI 滑块实时控制 Reachy Mini 机器人的头部、天线和身体姿态的控制面板。

## 功能特性

- **头部位置控制**: X (前后), Y (左右), Z (上下)
- **头部角度控制**: Roll (翻滚), Pitch (俯仰), Yaw (偏航)
- **天线控制**: 左天线、右天线角度
- **身体控制**: Body Yaw (身体偏航角)
- **实时响应**: 拖动滑块即可实时更新机器人姿态
- **一键重置**: 快速回到中心位置

## 运行方式

```bash
# 在项目根目录运行
uv run demo2_slider_control/slider_control.py

# 或者直接运行
python demo2_slider_control/slider_control.py
```

## 控制参数范围

| 类别 | 参数 | 范围 | 单位 | 说明 |
|------|------|------|------|------|
| **头部位置** | X | ±50 | mm | 前后位置，向前为正 |
| | Y | ±50 | mm | 左右位置，向左为正 |
| | Z | -30 ~ +80 | mm | 上下位置，向上为正 |
| **头部角度** | Roll | ±25 | 度 | 翻滚角，右倾为正 |
| | Pitch | ±35 | 度 | 俯仰角，抬头为正 |
| | Yaw | ±160 | 度 | 偏航角，左转为正 |
| **天线** | 左天线 | -80 ~ +80 | 度 | 左天线角度 |
| | 右天线 | -80 ~ +80 | 度 | 右天线角度 |
| **身体** | Body Yaw | -160 ~ +160 | 度 | 身体偏航角 |
| **运动参数** | Duration | 0.1 ~ 10 | 秒 | 运动时长（用于重置功能） |

## 技术说明

- 使用 `ReachyMini.set_target()` 实现实时控制（无插值，响应最快）
- 使用 `ReachyMini.goto_target()` 实现平滑运动（带插值，用于重置功能）
- 自动进行单位转换：界面使用度/毫米，SDK 使用弧度/米
- 基于 tkinter GUI，跨平台支持

## 依赖

- Python 3.8+
- reachy-mini SDK
- tkinter (通常 Python 自带)
